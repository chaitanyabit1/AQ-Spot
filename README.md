## Setup the codebase

**Install all the packages and dependencies:**

`npm install`

<br />

**Add the configuration keys:**

Copy the `.env-sample` file and name it `.env` file.

Open the `.env` file and add the configurations there.

<br />

## Run the app

**Run the app:**

`npm start`

<br />

**Prepare the build:**

`npm run build`
